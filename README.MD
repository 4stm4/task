##### создаем виртуальную среду

`python3 -m venv venv`

  

##### активируем среду

`source venv/bin/activate`

  

##### устанавливаем зависимости

`python3 -m pip install -r requirements.txt`

  

##### запускаем приложение

`python3 run.py`

  

##### запуск тестов

`python3 -m pytest --cov=app --cov-report term-missing`
